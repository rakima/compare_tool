# compare_tool v0.4.0 Release notes

JSON比較に対応し、Excel/CSVに加えてJSONファイル同士の差分も確認できるようになりました。JSON Path単位の差分一覧をExcelに出力し、JSON専用オプションでキー順や配列順序の扱いを調整できます。

## Highlights

- JSON同士の比較に対応
- JSON比較結果を `.xlsx` で出力
- JSON専用オプションを追加
- JSON読み込みエラーの説明を改善
- ファイル形式ごとにGUIオプション表示を切り替え
- アプリバージョン表示を `0.4.0` に更新

## Added

- `.json` vs `.json` の比較に対応しました。
  - JSONはJSON Path単位で比較します。
  - 比較結果は従来通り `.xlsx` で出力します。
  - 新JSONの内容は `JSON` シートに整形して出力します。
- JSONの差分検出に対応しました。
  - 値変更
  - キー追加
  - キー削除
  - 配列要素追加
  - 配列要素削除
- JSON専用オプションを追加しました。
  - オブジェクトのキー順を無視
  - 配列の順序を無視
- JSON比較結果のExcelに、JSON比較設定を記録するようにしました。
  - 文字コード
  - 比較位置
  - オブジェクトのキー順を無視
  - 配列順序を無視
- 手動確認用サンプル生成にJSONを追加しました。

## Improved

- GUIのオプション表示をファイル形式別Frameに分割しました。
  - Excel同士ではExcel/表形式オプションを表示します。
  - CSV同士ではCSV/表形式オプションを表示します。
  - JSON同士ではJSON専用オプションを表示します。
  - 関係ない形式のオプションはグレーアウトではなく非表示にします。
- JSON読み込みエラーの説明を改善しました。
  - JSON構文エラーの行・列を表示します。
  - UTF-8として読めない場合の対処を表示します。
  - ファイルアクセス失敗時の確認ポイントを表示します。
- `配列の順序を無視` がONのときの差分表示を改善しました。
  - 同じ要素は順序が違っても相殺します。
  - 旧JSONにだけ残る要素は削除として表示します。
  - 新JSONにだけ残る要素は追加として表示します。
- アプリのバージョン表示を固定バージョン定義から参照するようにし、ビルド済みアプリでも表示が古いバージョンに戻らないようにしました。

## Fixed

- GUIタイトルバー/ヘッダーのバージョン表示が古いインストール済みパッケージ情報を参照する可能性を修正しました。

## Known limitations

- `.xlsx` と `.csv`、`.csv` と `.json` のような異なる形式同士の比較には対応していません。
- `.xls` は変換層のみ追加済みで、実際の比較にはまだ対応していません。
- `.xlsm`、XMLには未対応です。
- JSONはUTF-8 / UTF-8 BOM付きのファイルを対象にしています。
- JSON配列のキー指定比較、配列LCS、JSON Lines、JSON5、スキーマ比較は未対応です。
- CSVの自動判定は推定です。意図と違う場合は文字コードや区切り文字を手動指定してください。
