# compare_tool v0.5.0 Release notes

XML比較に対応し、Excel/CSV/JSONに加えてXMLファイル同士の差分も確認できるようになりました。XPath風パスで差分を表示し、XML専用オプションや子要素の対応付け改善により、構造差分を確認しやすくしています。

## Highlights

- XML同士の比較に対応
- XML比較結果を `.xlsx` で出力
- XML専用オプションを追加
- XML子要素のLCS対応付けを追加
- `id` / `name` 属性によるXML要素対応付けを追加
- XML名前空間付き要素・属性の表示を改善
- XMLサンプルファイル生成に対応

## Added

- `.xml` vs `.xml` の比較に対応しました。
  - XMLはXPath風パス単位で比較します。
  - 比較結果は従来通り `.xlsx` で出力します。
  - 新XMLの内容は `XML` シートに整形して出力します。
- XMLの差分検出に対応しました。
  - 要素追加
  - 要素削除
  - テキスト変更
  - 属性追加
  - 属性削除
  - 属性値変更
- XML専用オプションを追加しました。
  - 属性順を無視
  - 空白のみテキストを無視
- XML比較結果のExcelに、XML読み込み/比較設定を記録するようにしました。
  - 文字コード
  - 比較位置
  - 属性順を無視
  - 空白のみテキストを無視
- 手動確認用サンプル生成にXMLを追加しました。

## Improved

- GUIで `.xml` を選択・ドラッグ&ドロップできるようにしました。
- XML同士を指定したときだけXML専用オプションFrameを表示するようにしました。
- XML子要素の見せ方を改善しました。
  - 同じ内容の子要素をLCSで対応付けます。
  - 途中に要素が追加・削除されても、後続要素がまとめて変更扱いになりにくくなりました。
- `id` / `name` 属性によるXML要素対応付けを追加しました。
  - 同じ親要素内でキーが一意な場合に対応付けます。
  - 並び替えがあっても同じ要素として比較しやすくなりました。
  - キーが重複する場合は誤対応を避けるため通常のLCS比較へ戻します。
- XML名前空間表示を改善しました。
  - 名前空間付き要素を `p:item` のように表示します。
  - 名前空間付き属性を `@p:code` のように表示します。
  - prefixのない既定名前空間はローカル名で表示します。
- XML読み込みエラーの説明を改善しました。
  - XML構文エラーの行・列を表示します。
  - UTF-8として読めない場合の対処を表示します。

## Known limitations

- `.xlsx` と `.csv`、`.csv` と `.json`、`.json` と `.xml` のような異なる形式同士の比較には対応していません。
- `.xls` は変換層のみ追加済みで、実際の比較にはまだ対応していません。
- `.xlsm` には未対応です。
- XMLはUTF-8 / UTF-8 BOM付きのファイルを対象にしています。
- XMLコメント、処理命令、DTD/XSD検証、名前空間URIの高度な正規化、任意キー指定による同名兄弟要素の比較、XML整形差分には未対応です。
- JSON Lines、JSON5、XML以外のマークアップ形式には未対応です。
- CSVの自動判定は推定です。意図と違う場合は文字コードや区切り文字を手動指定してください。
