# compare_tool v0.3.0 Release notes

CSV比較に対応し、Excel以外のファイル形式も比較できるようになりました。CSVの読み込み設定は自動判定を既定にし、比較結果Excelにも読み込み設定を記録します。

## Highlights

- CSV同士の比較に対応
- CSV文字コードの自動判定を追加
- CSV区切り文字の自動判定を追加
- CSV空行の無視オプションを追加
- CSV読み込み設定を比較結果Excelへ記録
- 比較オプションの前回値保存に対応
- `.xls` 対応に向けた変換層を追加

## Added

- `.csv` vs `.csv` の比較に対応しました。
  - CSVは内部的に `CSV` という1シートの表として扱います。
  - 比較結果は従来通り `.xlsx` で出力します。
- CSV文字コードを選択できるようにしました。
  - 自動
  - UTF-8 / UTF-8 BOM
  - Shift_JIS
- CSV文字コードの `自動` 判定を既定にしました。
  - UTF-8 BOMを優先します。
  - BOMがない場合はUTF-8、Shift_JISの順に読み取りを試します。
- CSV区切り文字を選択できるようにしました。
  - 自動
  - カンマ
  - タブ
  - セミコロン
- CSV区切り文字の `自動` 判定を既定にしました。
  - 先頭行を候補ごとに読み比べ、列数が多く安定している区切り文字を選びます。
- CSV空行を無視するオプションを追加しました。
  - 既定では空行を無視します。
  - 空行自体を差分として見たい場合はOFFにできます。
- CSV比較結果のExcelに、読み込み設定を記録するようにしました。
  - 文字コード
  - 区切り文字
  - 空行を無視
- 比較設定の前回値保存に対応しました。
  - 比較オプション
  - 表示方法
  - 比較方式
  - キー列
  - CSV読み込み設定
- `.xls` 対応に向けた変換層を追加しました。
  - 現時点では `.xls` の実変換は未実装です。

## Improved

- GUIで指定ファイルの形式に応じて、Excel専用/CSV専用オプションの有効・無効を切り替えるようにしました。
  - CSV同士では `数式を比較` を無効化します。
  - Excel系同士ではCSV専用オプションを無効化します。
- CSV読み込みエラーの説明を改善しました。
  - 文字コード不一致
  - 自動判定失敗
  - CSV形式の破損
  - ファイルアクセス失敗
  - 区切り文字指定エラー
- 手動確認用サンプル生成にCSVを追加しました。
  - UTF-8カンマ区切りCSV
  - Shift_JIS + タブ区切りCSV

## Known limitations

- `.xlsx` と `.csv` のような異なる形式同士の比較には対応していません。
- `.xls` は変換層のみ追加済みで、実際の比較にはまだ対応していません。
- `.xlsm`、JSON、XMLには未対応です。
- CSVの自動判定は推定です。意図と違う場合は文字コードや区切り文字を手動指定してください。
